# pekzep_markov

直前の字が0.9票、二つ前の字が0.06票、三つ前の字が0.02票、四つ前の字が0.01票を入れます。


## Environment variable

Requires you to set `DISCORD_NOTIFIER_TOKEN` in the env var for the discord notifier to work.

## Running Locally

```sh
$ npm install
$ npm run build
$ npm start
```

## To compile index.ts

```sh
$ npm install
$ tsc
```
