# pekzep_markov

直前の字が0.5票、二つ前の字が0.3票、三つ前の字が0.15票、四つ前の字が0.05票を入れます。


## Environment variable

Requires you to set `DISCORD_NOTIFIER_TOKEN` in the env var for the discord notifier to work.

## Running Locally

```sh
$ npm install
$ npm run build
$ npm start
```

## To compile index.ts

```sh
$ npm install
$ tsc
```
